# 算法与数据结构

## 一、数据结构基础

### 1.1 数组与链表
- 数组：连续内存、随机访问 O(1)、插入删除 O(n)
- 链表：单链表、双链表、循环链表
- 经典题：反转链表、合并有序链表、环形链表检测

### 1.2 栈与队列
- 栈：LIFO、应用：括号匹配、单调栈
- 队列：FIFO、双端队列 deque
- 优先队列（堆）：最大堆、最小堆
- 经典题：有效的括号、最小栈、滑动窗口最大值

### 1.3 哈希表
- 原理：哈希函数、冲突解决（链地址法、开放寻址法）
- 常见应用：两数之和、字母异位词、LRU 缓存

### 1.4 树
- 二叉树：前序、中序、后序遍历、层序遍历
- 二叉搜索树（BST）：查找、插入、删除
- 平衡二叉树：AVL 树、红黑树基本概念
- 堆：建堆、堆排序
- 经典题：最大深度、对称二叉树、路径总和

### 1.5 图
- 表示方式：邻接矩阵、邻接表
- 遍历：BFS（广度优先）、DFS（深度优先）
- 最短路径：Dijkstra、Floyd
- 拓扑排序：课程表问题
- 并查集：连通分量

## 二、常见算法

### 2.1 排序算法
- 冒泡排序：O(n²)、稳定
- 选择排序：O(n²)、不稳定
- 插入排序：O(n²)、稳定、小数据量高效
- 快速排序：O(nlogn) 平均、partition 思想
- 归并排序：O(nlogn)、稳定、分治思想
- 堆排序：O(nlogn)、不稳定
- 计数排序/桶排序/基数排序：非比较排序

### 2.2 查找算法
- 二分查找：标准模板、变体（左边界、右边界）
- 二分答案：搜索空间上二分

### 2.3 双指针
- 快慢指针：链表环检测
- 对撞指针：两数之和、盛最多水的容器
- 滑动窗口：无重复字符的最长子串、最小覆盖子串

### 2.4 递归与回溯
- 递归三要素：终止条件、递推公式、返回值
- 回溯模板：选择-递归-撤销
- 经典题：全排列、子集、N 皇后、组合总和

### 2.5 动态规划
- 核心思想：最优子结构、重叠子问题、状态转移方程
- 解题步骤：定义状态 → 写转移方程 → 确定初始值 → 确定遍历顺序
- 经典题：
  - 入门：爬楼梯、最大子数组和、打家劫舍
  - 背包问题：0-1 背包、完全背包
  - 字符串：最长公共子序列、编辑距离
  - 股票问题：买卖股票的最佳时机系列

### 2.6 贪心算法
- 核心：局部最优 → 全局最优
- 经典题：跳跃游戏、分发饼干、区间调度

## 三、刷题策略

### 3.1 学习路线
1. 先掌握基本数据结构（数组、链表、栈、队列、哈希表）
2. 学习二叉树的遍历和基本操作
3. 掌握排序和查找算法
4. 重点突破双指针和滑动窗口
5. 系统学习动态规划
6. 图论和高级数据结构按需学习

### 3.2 面试高频考点
- Top K 问题：堆、快速选择
- LRU 缓存：哈希表 + 双向链表
- 二叉树序列化与反序列化
- 字符串匹配：KMP 算法
- 前缀和与差分数组
